:::


## Prerequisites


- API Keys and SDK version names for mobile integration
- Server Keys for back-end integration 

:::tip Note
The following keys will be shared over an email
- `ACCESS_KEY`
- `SECRET_KEY`
- `LENDING_SDK_VERSION`
- `DC_SDK_VERSION`
- `COMMON_SDK_VERSION`
- `LOGGER_SDK_VERSION`
- `BC_SDK_VERSION`
- `PLATFORM_NAME`
- `SERVER_API_KEY` 
:::
