Users' credential is verfified electonically using Aadhar or PAN based substantiation.

We ask user to upload their selfie, PAN, and Aadhar(as address proof) and GST(if required). User needs to enter OTP sent on phone number registered with Aadhar, then the eKYC process is completed.
