Offer Salary advances to employees or borrowers via your HR-Tech or Employee Portal. It is a fully managed end-to-end journey. 

## About Salary Advance

* The loan amount will be less than their salary

**User is shown a hook offer to apply for a loan** - The first step in the user journey is applying for loan

**Bank Details** - Then user is asked for bank account details and we do penny drop to verify.


**E-KYC**
Following this the user completes his eKYC by submitting his

* Address proof  (Aadhaar or Passport)
* Selfie 

**Sets up ENACH** - User sets up autopay. There are 2 ways of doing so depending on the bank - 
* E-NACH
* Manual NACH

**Completes the application process** - User signs a loan agreement and completes the application journey.
